# Cricket Project (Personal Website)

A cricket-themed website with player pages + a T20 World Cup section.
Includes a Node.js + MongoDB backend to store match data.

## Tech
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- DB: MongoDB Atlas

## Run locally
### Backend
cd t20wc-backend
npm install
npm start

### Frontend
Open the project in VS Code and use Live Server.
